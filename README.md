# Insight Weaver

> AI-powered document analysis platform that weaves raw data into actionable business intelligence.

![build](https://img.shields.io/badge/build-passing-brightgreen) ![license](https://img.shields.io/badge/license-MIT-blue) ![stack](https://img.shields.io/badge/stack-TypeScript%20%7C%20Vite%20%7C%20Supabase-blueviolet)

## 📖 The Story
> *"The reports were massive — hundreds of pages of raw data with no synthesis. Insight Weaver consumed them, cross-referenced patterns, and surfaced the three critical findings buried on page 247. The executive summary wrote itself."*

## ✨ Features
| Feature | Description |
| :--- | :--- |
| **Document Ingestion** | Upload PDFs, CSVs, and text files for automated analysis |
| **Pattern Recognition** | NLP-powered extraction of trends, anomalies, and key metrics |
| **Executive Summaries** | Auto-generated summaries with confidence scoring |
| **Interactive Dashboard** | Drill-down visualizations of extracted insights |
| **Export & Share** | Generate branded reports in PDF or presentation format |

## 🛠 Tech Stack
- **Frontend**: Vite, React, TypeScript
- **UI**: Tailwind CSS, Shadcn UI
- **Backend**: Supabase
- **AI**: OpenAI API

## 🚀 Quick Start
```bash
git clone https://github.com/ayushjhaa1187-spec/insight-weaver.git
cd insight-weaver
npm install
npm run dev
```
